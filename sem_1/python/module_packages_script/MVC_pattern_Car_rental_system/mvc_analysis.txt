what is mvc of my project :- car rental systm

| Layer          | Responsibility        | From Your SRS                      |
| -------------- | --------------------- | ---------------------------------- |
| **Model**      | Business logic + data | Car, User, Booking, Payment        |
| **View**       | UI / Output           | Booking confirmation, availability |
| **Controller** | Handles flow          | Booking logic, validation          |
