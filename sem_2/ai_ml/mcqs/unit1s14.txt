🔥 MCQ 2 (Very Important)

Which of the following statements is TRUE?

Standardization makes mean = 0
Standardization makes variance = 1
Min-Max scaling is robust to outliers

A 1 only
B 1 and 2
C 2 and 3
D 1,2,3

✅ Answer: B

💡 Trap: Min-Max is NOT robust to outliers

🔥 MCQ 3 (Numerical Thinking)

If a feature has zero variance (all values same), what happens during standardization?

A Values become 0
B Values become 1
C Division by zero occurs
D Values remain unchanged

✅ Answer: C

💡 Very common GATE trap.

B Random Forest

✅ Answer: C

🔥 MCQ 5 (Tricky)

Which statement is FALSE?

A Standardization centers data
B Min-Max scaling depends on min & max
C Binarization uses mean as threshold
D Feature scaling improves convergence

✅ Answer: C

💡 Binarization uses threshold, not necessarily mean.

🧠 FINAL EXAM STRATEGY

When solving questions:

Step 1: Identify
Scaling OR Selection?
Step 2: If scaling
Keyword	Answer
Range [0,1]	Min-Max
Mean = 0	Standardization
0/1	Binarization
Step 3: If selection
Keyword	Answer
Classification	chi2
Regression	f_regression
